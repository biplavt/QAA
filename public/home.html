API

GET /v1/QAA/testData

Get test Summary
JSON Structure
[ { "testID": Test Id, "employeeID": Employee Id, "EmployeeName": Employee Name, "Location": Location, "ProductID": Product Id, "Product": Product Name, "QuantityInspected": Quantity Inspected, "InspectionDate": Inspection Date", "Status": Status of test }, { ... }, ... ]

POST /v1/QAA/testData

POST test Summary
JSON Structure for Body
{ "testData":{ "employeeID": Employee Id, "locationID": Location Id, "modelID": Model Id, "Qty": Quantity } }

GET /v1/QAA/testDetailByID/:testId

Get test detail by testId
JSON Structure
[ { "TestCase": Test Case Number, "ProductID": Product Id, "WorkCell": Work Cell Number(Id), "Qty": Quantity, "Criteria": Criteria of testing, "Ideal": Ideal value , "rLow": Low Value, "rHigh": High Value, "Actual": Actual Value, "Status": Status of Test },

POST /v1/QAA/TestLine

Post testline
JSON Structure
{ "testData": [ { "TestCase": Test ID, "criteriaID": Criteria Id, "rangeID": Range Id, "testData":91 }, { ... }, ... ] }

GET /v1/QAA/Location

Get location
JSON Structure
[ { "locationID": Location Id, "locationName": Location Name } ]

GET /v1/QAA/Models

Get Model
JSON Structure
[ { "modelID": Model Id, "modelName": Model Name }, { ... }, ... ]

GET /v1/QAA/allCriteria

Get all Criteria
JSON Structure
[ { "criteriaID": Criteria Id, "criteriaName": Criteria Name }, { ... }, ... ]

GET /v1/QAA/rangeValuesForRangeID/:rangeId

Get range values for Range Id
JSON Structure
[ { "rangeIdeal": Ideal Value, "rangeLow": Low Value, "rangeHigh": High Value } ]
